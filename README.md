# Разработка интеллектуальных систем (Development of Intelligent Systems)

Репозиторий содержит коллекцию программных моделей и алгоритмов искусственного интеллекта, реализованных на языке Python. Проект охватывает ключевые подходы к созданию интеллектуальных систем: от классических экспертных моделей до гибридных нейро-нечетких сетей и эволюционных вычислений.

## 📋 Содержимое репозитория

В проекте представлены следующие модули и модели:

* **`ANFIS_model.py`** — Адаптивная нейро-нечеткая система вывода (ANFIS).
* **`CBR_model.py`** — Система рассуждений на основе прецедентов (Case-Based Reasoning).
* **`fuzzy_logic_model.py`** — Модели нечеткой логики (Fuzzy Logic).
* **`genetic_model.py`** — Генетический алгоритм (Genetic Algorithm) для задач оптимизации.
* **`decision_tree_model.py`** — Дерево решений (Decision Tree) для классификации и регрессии.
* **`product_model.py`** — Продукционная модель представления знаний (правила «Если ..., то ...»).
* **`frame_model.py`** — Фреймовая модель представления знаний.
* **`semantic_model.py`** — Семантическая сеть для структурирования знаний.
* **`adaptive_mpdel.py`** — Адаптивная модель управления или прогнозирования.
* **`analitic_model.py`** — Аналитические методы оценки и расчета.
* **`support_model.py`** — Вспомогательные скрипты и утилиты для поддержки работы основных моделей.

## 🛠️ Технологический стек

* **Язык программирования:** Python 3.x
* **Основные библиотеки:** `numpy`, `scipy`, `scikit-learn`, `matplotlib` (дополните список при необходимости)

## 🚀 Начало работы

### Требования
Убедитесь, что у вас установлен Python версии 3.8 или выше.

### Клонирование репозитория
```bash
git clone https://github.com
cd development-of-intelligent-systems
```

### Запуск моделей
Каждый скрипт является самостоятельным модулем. Вы можете запустить интересующую модель напрямую через терминал. Например, для запуска генетического алгоритма:
```bash
python genetic_model.py
```
